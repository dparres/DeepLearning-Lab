# CIFAR-10 Objectives

This assignment has a maximum weight of 2 points. This score is distributed based on the accuracy achieved on the MNIST test set. To attain the maximum score, a minimum accuracy of 92.0% must be reached.

| Accuracy | Points |
| --------------- | --------------- |
| 86.0%  | 0.5   |
| 88.0%  | 0.5   |
| 90.0%  | 0.5   |
| 92.0%  | 0.5   |
